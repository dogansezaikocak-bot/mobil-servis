# Ekzen Servis Takip V3.3.11 - Ertele Takvim Fix

## Bu sürümde yapılan düzeltme

- Mobil servis detayında **Ertele** butonu artık eski tarayıcı yazı kutusu açmaz.
- Bottom sheet içinde görünür **Yeni Tarih** alanı açılır.
- Tarih telefonun/tarayıcının normal takvim alanından seçilir.
- **Kaydet** ile servis seçilen tarihe taşınır.
- **Vazgeç** ile işlem kapatılır.
- Seçilen tarihe geçilerek ilgili servis listede gösterilir.

## Yükleme

ZIP içindeki dosyaları `mobil-servis` reposunun kök dizinine yükleyip **Commit changes** yapın.
