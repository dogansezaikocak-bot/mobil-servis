# Ekzen Servis Takip V3.3.8

Mobil arayüz devam sürümü.

## Eklenenler
- Yeni fiş açma ekranı mobilde alttan açılan, tek elle kullanılabilir forma çevrildi.
- Servis detay ekranı bottom sheet olarak zenginleştirildi.
- Servis detayında müşteri, telefon, adres, cihaz, şikayet, kaynak, durum, ücret ve tarih/saat alanları eklendi.
- Hızlı aksiyon butonları eklendi: Ara, WhatsApp, Harita, Not, Fotoğraf, Düzenle, Ertele, Fişi Kapat.
- Bottom sheet açılış animasyonu ve mobil taşma düzeltmeleri yapıldı.

## Not
Mevcut kayıt mantığı ve localStorage veri yapısı korunmuştur.
