# Ekzen Servis Takip V10.0

- Dağıtım ve Servisler aynı canlı bulut mantığını kullanır.
- Teslim fotoğrafı kamera olayı tek kez tetiklenir.
- Her basışta yalnızca bir fotoğraf çekilir ve kaydedilir.
- Çoklu alanlara yeni fotoğraf eklemek için düğmeye tekrar basılır.
- Tekli alanlarda Tekrar Çek mevcut fotoğrafı değiştirir.
- Fotoğraf paneli açıkken canlı senkronizasyon paneli titretmez.
- D1, R2, Firebase canlı kayıt ve mevcut dağıtım verileri korunur.
- Worker kodunu yeniden yüklemek gerekmez.
